# vim

### basic operation

* `i` 
* `x`
* `:wq`
* `dd`
* `p`
* `hjkl`

### faster operation
#### insert
* `a`
* `o`
* `O`
* `cw`

#### cursor move
* `0`
* `^`
* `$`
* `g_`
* `/pattern`

#### copy/paste
* `P`
* `yy`

#### undo/redo
* `u`
* `<Ctrl-r>`

#### open/save/quit/change file
* `:e <path>`
* `:saveas <path>`
* `:q!`
* `:bn` and `:bp`

### more useful operation
* `.`
* `N<command>`
* `NG`
* `G`
* `gg`
* `w`
* `e`
* `%`
* `*` and `#`
* `<start positon><command><end position>`

eg. `0y$`  

`0` move to the beginning of the line  

`y` copy

`$` copy until the last word of the line
