Markdown
========
### 目的
使用文本的方式更简单的写出HTML页面

### 主要格式
#### 标题
使用#来代表不同的标题标签
#### 引用
使用’> 引用‘来实现
> 引用
#### 强调
使用```**bold**```或```__bold__```来实现**加粗**

使用```*em*```或```_em_```来实现*倾斜*

使用```~~delete~~```来实现~~删除~~

使用``` `linux` `Markdown` `git` ```来实现`linux` `Markdown` `git`

#### 并列
##### 无序排列
```
+ line1
+ line2
```
或
```
- line1
- line2
```
或
```
* line1
* line2
```
实现
+ line1
+ line2

- line1
- line2

* line1
* line2
##### 有序排列
```
1. line1
1. line2
1. line3
	1. line31
	1. line32
	1. line33
```
1. line1
1. line2
1. line3
	1. line31
	1. line32
	1. line33
#### 表格
使用
```
| title1 | title2 | title3 |
| :--- | :--: | ---: |
| left | middle | right |
| l | m | r |
```

| title1 | title2 | title3 |
| :--- | :--: | ---: |
| left | middle | right |
| l | m | r |
